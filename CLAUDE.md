## Projet: Gestionnaire de Tâches

Ceci est un projet Python **basique pour lycéen débutant** qui apprend les fondamentaux de Python.

### Structure du Projet

- `main.py`: Programme principal avec le gestionnaire de tâches
- `taches.txt`: Fichier généré automatiquement pour sauvegarder les tâches (ignoré par git)
- `README.md`: Documentation simple du projet

### Comment lancer le programme

```bash
python main.py
```

### Compétences couvertes

- Listes Python
- Lecture/écriture de fichiers
- Boucles et conditions
- Fonctions
- Gestion basique des erreurs

### Notes pour le développement

- Le code est volontairement **simple et lisible** pour un débutant
- Les commentaires sont en français pour la clarté
- Les noms de variables et fonctions sont descriptifs
- Pas de bibliothèques externes nécessaires

### Améliorations futures possibles

Ces améliorations pourraient être faites après maîtrise des bases:
- Ajouter une date/heure aux tâches
- Marquer les tâches comme "complétées" au lieu de les supprimer
- Utiliser une base de données au lieu d'un fichier txt
- Ajouter une interface graphique (GUI)
